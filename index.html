Sobre o Projeto

Contexto:
Quando se trabalha com grandes volumes de dados introduzidos manualmente, é comum surgirem erros de escrita. Estes erros dificultam a análise estatística, uma vez que, sem um padrão definido, torna-se extremamente trabalhoso consolidar e validar a informação de forma eficiente. Em contextos com milhões de registos, esta tarefa torna-se impraticável de ser realizada manualmente, algo que tem acontecido até agora.
Este projeto foi proposto pelo Porto de Sines à Universidade de Aveiro, com o objetivo de desenvolver um sistema capaz de corrigir automaticamente tabelas de dados, recorrendo a técnicas de mineração de texto.

Solução:
A solução desenvolvida baseia-se numa abordagem automática para normalização e agrupamento de nomes, utilizando técnicas de processamento de linguagem natural e aprendizagem automática.
O sistema recebe como entrada uma tabela de dados e, opcionalmente, uma tabela de dados Standard. Caso seja oferecida a tabela Standard ao sistema, este cria um Ground Truth para servir de apoio ao algoritmo de clustering.
O processo começa com uma etapa de pré-processamento, onde os nomes são limpos e normalizados, removendo ruído, o que garante que nomes semanticamente semelhantes sejam tratados uniformemente. De seguida, os nomes são convertidos em vetores numéricos (embeddings) com o modelo Sentence Transformer. Com base nestas representações vetoriais, é aplicado o nosso algoritmo de clustering para agrupar automaticamente nomes semelhantes.
Caso exista um ground truth, a qualidade dos agrupamentos é avaliada com métricas como precision, recall e F1-score.
